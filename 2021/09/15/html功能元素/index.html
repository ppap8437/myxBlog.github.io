前言

​ 从最简单的网页文本显示，再到css修改文本样式、添加表单提交、css动画、添加各种js事件，功能元素也是十分重要的。

功能元素

1.列表标签

​ 无序列表 ul>li ，数据没有先后顺序

​ 作用：给一堆数据添加列表语义，也就是标签先告诉搜索引擎，再告诉浏览器这一堆数据是一个整体

​ ul type=“disc,circle,item,square”
​ disc 默认值 实心圆
​ circle 空心圆
​ item 实心圆
​ square 实心方块

关于无序列表的注意点：

​ 1.ul和li标签是一个整体，不建议在在ul标签中包含其他标签

​ 2.ul标签是用来给一堆数据添加列表语义的，而不是用来给他们添加小圆点。

​ 有序列表 ol>li

​ 作用： 给一堆数据添加列表语义, 并且这一堆数据中所有的数据都有先后之分

​ type=”1,a,A,i,I”

​ 1 默认值。数字有序列表
​ a 按字母顺序排列的有序列表，小写排序
​ A 按字母顺序排列的有序列表，大写排序
​ i 罗马字母，小写排序
​ I 罗马字母，大写排序

​ 定义列表

​ 作用：给一堆数据添加列表语义，先通过dt标签定义列表中的所有标题, 然后再通过dd标签给每个标题添加描述信息

关于无序列表的注意点：

​ 1.dl和dt、dd是一个整体，不建议在dl标签中放其他标签

​ 2.和li标签一样, 当需要丰富界面时, 我们可以在dt和dd标签中继续添加其它标签

2.表格标签

​ 作用：给一堆数据添加语义

表格标签样式：

​ 其实表格标签中的tr标签代表整个表格中的一行数据, 也就是说一对tr标签就是表格中的一行

​ 其实表格标签中的td标签代表表格中一行中的一个单元格, 也就是说一对td标签就是一行中的一个单元格

宽高和高度属性设置

​ 可以给table标签和td标签使用

​ 1.表格的宽度和高度默认是按照内容的尺寸来调整的, 也可以通过给table标签设置width/height属性的方式来手动指定表格的宽度和高度.

​ 2.如果给td标签设置widht/height属性, 会修改当前单元格的宽度和高度, 不会影响整个表格的宽度和高度

水平对齐和垂直对齐的属性

​ 其中水平对齐可以给table标签和tr标签和td标签使用.

​ 垂直对齐只能给tr标签和td标签使用.

​ 1.给table标签设置align属性, 可以控制表格在水平方向的对齐方式.

​ 2.给tr标签设置align属性, 可以控制当前行中所有单元格内容的水平方向的对齐方式.

​ 3.给td标签设置align属性, 可以控制当前单元格中内容在水平方向的对齐方式.（如果td中设置了align属性, tr中也设置了align属性, 那么单元格中的内容会按照td中设置的来对齐）

​ 4.给td标签设置valign属性, 可以控制当前单元格中的内容在垂直方向的对齐方式.

​ align: left、center 、right

​ valign: top、mid、bottom

​ 外边距内边距

​ cellspacing外边距 单元格和单元格之间的距离 默认2px；

​ cellpadding内边距 单元格的边框和文字之间的间隙 默认1px；

单元格合并

​ 1.水平方向上的单元格合并（colspan标签）

​ 2.垂直方向上的单元格合并（rowspan标签）

​ 一定要记住单元格合并永远都是向后或者向下合并, 而不能向前或者向上合并.

3.form表单

​ form表单就是专门用来收集用户信息的

表单格式：

​ 注意:表单元素一定要写在表单中

常见表单元素

input标签
input标签有一个type属性, 这个属性有很多类型的取值,取值的不同就决定了input标签的功能和外观的不同
1.明文输入框

​ name:表单提交时，会作为参数的名称，结合输入的值进行匹配，组成 参数名=参数值的结构
​ placeholder:在用户输入之前给用户相应的提示，以减少出错的情况

​ 2.暗文输入框（多用于密码输入）

​ 3.单选框

​ 注意点：

​ 1.默认情况下单选框不会互斥, 要想单选框互斥那么必须给每一个单选框标签都设置一个name属性, 然后name属性还必须设置相同的值。

​ 2.要想让单选框默认选中某一个, 那么可以给input标签添加一个checked属性。

​ 3.单选框的value值，才是我们最终需要的值，并不是单选框显示的值。

​ 4.多选框

​ 5.提交按钮

​ 作用：

​ 将表单中已经填写好的数据, 提交到远程服务器

​ 注意点：

​ 1.需要给form表单添加一个action的属性, 通过这个action属性指定需要提交到的服务器地址

​ 2.需要提交到服务器的表单元素添加一个name属性

​ 6.普通按钮

​ 作用：配合JS完成一些操作

​ 7.图片按钮

​ 作用：配合JS完成一些操作

​ 8.重置按钮

​ 作用：用于清空表单中已经填写好的数据

​ 注意点：如果想想改重置按钮默认的按钮标题可以通过value属性来修改

​ 9.隐藏域

​ 作用 : 配合提交按钮将一些数据默默的悄悄的提交到服务器。

4.label标签

​ 默认情况下文字和输入框是没有关联关系的, 也就是说点击文字输入框不会聚焦, 如果想点击文字时让对应的输入框聚焦, 那么就需要让文字和输入框进行绑定。要想让输入框和文字绑定在一起, 那么我们可以使用label标签

​ label标签的用法：

​ 1.将文字利用label标签包裹起来

​ 2.给输入框添加一个id属性

​ 3.在label标签中通过for属性和输入框中的id进行绑定即可

5.select标签

​ 作用：select 元素可创建单选或多选菜单。

​ 用法：<select&> 元素中的

​ 注意点：

​ 1.下拉列表不能输入内容, 但是可以直接在列表中选择内容

​ 2.可以通过给option标签添加一个selected属性来指定列表的默认值

​ 3.可以通过给option标签包裹一层optgroup标签来给下拉列表中的数据分类

​ html5 新增的属性

属性 值 描述
autofocus autofocus 规定在页面加载后文本区域自动获得焦点。
disabled disabled 规定禁用该下拉列表。
form form_id 规定文本区域所属的一个或多个表单。
multiple multiple 规定可选择多个选项。
name name 规定下拉列表的名称。
required required 规定文本区域是必填的。
size size 规定下拉列表中可见选项的数目。

6.textarea标签

​ 文本区中可容纳无限数量的文本，其中的文本的默认字体是等宽字体（通常是 Courier）。

​ 可以通过 cols 和 rows 属性来规定 textarea 的尺寸，不过更好的办法是使用 CSS 的 height 和 width 属性。

​ 作用: 定义一个多行输入框

7.fieldset组件

​ fieldset 元素可将表单内的相关元素分组。

​ filedset标签没有必需的或唯一的属性。

8.HTML5新增的表单元素

1.datalist标签

​ 定义：datalist 标签定义选项列表。请与 input 元素配合使用该元素，来定义 input 可能的值。

​ 如何给输入框绑定待选列表

​ 1.搞一个输入框
​ 2.搞一个datalist列表
​ 3.给datalist列表标签添加一个id
​ 4.给输入框添加一个list属性,将datalist的id对应的值赋值给list属性即可

2.progress进度条

​ 用法：标签标示任务的进度（进程）

​ progress表示任务的完成情况，常用于进度条
​ max 定义进度元素所要求的任务的工作量，默认值为1
​ value 定义已经完成的工作量，如果max值为1，该值必须是介于0~1之间的小数。
